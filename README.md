# Digital Twin Management Dashboard

## 🚀 Overview
This project is a **Digital Twin** implementation featuring a real-time monitoring dashboard and a predictive backend engine. It combines a modern **React + Tailwind** frontend with a **Python-based ML simulation** backend.

## 🏗 Project Architecture
- **Frontend**: React 18, Vite, TailwindCSS, and Lucide Icons.
- **Backend**: Python-based simulation engine with machine learning models for predictive maintenance and performance modeling.

## 📁 Structure
- `/frontend`: The interactive dashboard for visualizing twin status and performance metrics.
- `/backend`: The core logic, including simulators and predictive models.

## 🏃 Getting Started

### 1. Backend (Python)
```bash
cd backend
python main.py
```

### 2. Frontend (React)
```bash
cd frontend
npm install
npm run dev
```

## ✨ Key Features
- **Real-time Synchronization**: Frontend reflects the simulated state of the digital twin.
- **Predictive Analytics**: ML models anticipate system failures and performance drops.
- **Modern UI/UX**: Clean, responsive dashboard designed for industrial monitoring.
