# 🏭 Predictive Digital Twin v2.0 Pro

A premium, AI-driven Industrial IoT monitoring system with advanced machine learning and real-time telemetry.

## 🛠️ Quick Start

### 1. Backend Setup
1. **Run Backend:**
   ```powershell
   cd backend
   pip install -r requirements.txt
   python main.py
   ```

### 2. Frontend Setup
1. **Run Dev Server:**
   ```powershell
   cd frontend
   npm run dev
   ```
2. **Access Dashboard:** `http://localhost:5173`

---

## 🌟 Key Features
- **LSTM + Self-Attention:** Temporal PER prediction (PyTorch).
- **SVM (RBF Kernel):** Non-linear failure classification.
- **Isolation Forest:** Unsupervised anomaly detection.
- **Gradient Boosting:** Ensemble health regression.
- **React Dashboard:** Real-time monitoring, alerts, charts, and node controls.
